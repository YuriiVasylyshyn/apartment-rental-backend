## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript project for managing apartment rentals.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# test coverage
$ npm run test:cov
```
